# Respiratory Care — Starter Website

This repository contains a simple Bootstrap-based static site scaffold for a respiratory care website. It includes:

- index.html (home)
- about.html
- services.html
- contact.html
- assets/css/style.css

Setup
1. This content was pushed to branch `website-starter` and to `main`.
2. To publish with GitHub Pages, merge the branch into the default branch (main) -- the files are already on main.
3. Enable Pages in repository settings (Settings → Pages → Source: Branch: main / Root). Wait a few minutes for deployment.

Customize
- Edit the HTML files to add real contact information, images, and logos.
- Replace the mailto form with a form service (Formspree, Netlify Forms) or a server endpoint for real submissions.
