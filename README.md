# 💍 Interactive Wedding Invitation

A beautiful, interactive, and fully customizable wedding invitation template. Light-themed design with smooth animations and modern UI.

## ✨ Features

- 🎨 **Light Color Scheme** - Beautiful pastel gradients (pink, purple, white)
- ⚡ **Smooth Animations** - Fade-in effects and hover transitions
- 📱 **Responsive Design** - Works perfectly on mobile and desktop
- 🎭 **Interactive Elements** - Modal forms and hover effects
- 💬 **Attendance Forms** - Confirm or decline with guest details
- 🖼️ **Gallery Section** - Interactive celebration highlights
- 🔧 **Easy to Customize** - Simple template for any event

## 📁 Files

- **index.html** - Mohamed & Shahenda's wedding invitation
- **template.html** - Blank template for customization
- **README.md** - This file

## 🚀 Quick Start

### Using the Invitation
1. Open `index.html` in a web browser
2. Click "Confirm Attendance" or "Decline" buttons
3. Fill in your details in the modal forms
4. Submit your response

### Creating Your Own Invitation
1. Copy or fork this repository
2. Open `template.html` in a text editor
3. Replace the placeholder text:
   - `Couple Name Here` → Your names
   - `DD/MM/YYYY` → Your date
   - `HH:MM` → Your time
   - `City, Country` → Your location
4. Save and open in browser
5. Deploy to GitHub Pages (see below)

## 🌐 Deploy to GitHub Pages

1. Go to repository Settings → Pages
2. Select "main" branch as source
3. Click Save
4. Your invitation will be live at: `https://username.github.io/repository-name`
